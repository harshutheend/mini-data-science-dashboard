# Mini Data Science Dashboard

## Overview

This project is a simple Data Science Dashboard built using Python.

## Features

- Load CSV dataset
- Summary statistics (Mean, Median, Std Dev)
- Correlation Matrix
- Bar Chart Visualization
- Histogram Visualization
- Interactive Menu Dashboard

## Tech Stack

- Python
- NumPy
- Matplotlib

## How to Run

1. Install dependencies:
   pip install numpy matplotlib
2. Run:
   python main.py
